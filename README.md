#  Vehicle Service Center Management System (Web & Mobile)

A full-stack web and mobile application developed for **Ananda Auto Motor Service, Kurunegala**, to digitalize and streamline vehicle service center operations. The system provides customers with an easy appointment booking experience while enabling workshop staff to efficiently manage daily operations through a real-time admin dashboard.

---

# 📌 Project Overview

Ananda Auto Motor Service operates with **10 vehicle service slots**.  
This system was designed to reduce manual record keeping and improve customer experience by providing:
- Online booking
- Service tracking
- Business analytics

---

# 🚨 Problem Statement

Many vehicle service centers still rely on manual systems (paper records, phone calls, Excel) to manage bookings and customer data, which leads to:
- Booking conflicts
- Missed appointments
- Poor service history tracking

At **Ananda Auto Motor Service**, managing only **10 service slots** manually caused delays, errors, and communication issues.

The system was built to **digitize and automate bookings**, improve scheduling efficiency, and provide real-time access to appointments and service records.
